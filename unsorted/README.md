# Unsorted

Temporary workspace for interview prep sessions before organizing into topic folders.

-   `interview_prep/` - Current active practice session
-   `archive/` - Completed sessions (moved here before organizing into topics)

## Workflow

1. Solve problems in `interview_prep/YYYY_MM/`
2. When done with session, move to `archive/YYYY_MM/`
3. Organize solutions from archive into topic folders (arrays_and_hashing, trees, etc.)
